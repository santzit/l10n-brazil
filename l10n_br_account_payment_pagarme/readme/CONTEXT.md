**Português**

Este módulo foi desenvolvido para atender a necessidade de integração com o gateway de pagamento Pagar.me no contexto brasileiro, fornecendo:

- **Integração Completa**: API v5 do Pagar.me com todas as funcionalidades principais
- **Múltiplos Métodos**: Cartões, boletos, PIX com configurações específicas
- **Segurança**: Criptografia, validação de webhooks, compliance PCI DSS
- **Automação**: Webhooks para atualizações automáticas de status
- **Flexibilidade**: Split de pagamentos, anti-fraude, parcelamento

**English**

This module was developed to meet the need for integration with the Pagar.me payment gateway in the Brazilian context, providing:

- **Complete Integration**: Pagar.me API v5 with all main functionalities
- **Multiple Methods**: Cards, boletos, PIX with specific configurations  
- **Security**: Encryption, webhook validation, PCI DSS compliance
- **Automation**: Webhooks for automatic status updates
- **Flexibility**: Payment splitting, anti-fraud, installments
