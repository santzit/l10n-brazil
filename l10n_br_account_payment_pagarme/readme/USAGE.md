**Português**

Após configurar o módulo:

1. **Criar Faturas**: Use modos de pagamento configurados com Pagar.me
2. **Processar Pagamentos**: Confirme ordens de pagamento para enviar ao Pagar.me
3. **Acompanhar Status**: Webhooks atualizam automaticamente o status
4. **Gerenciar Transações**: Use o wizard para capturar, cancelar ou estornar

**English**

After configuring the module:

1. **Create Invoices**: Use payment modes configured with Pagar.me
2. **Process Payments**: Confirm payment orders to send to Pagar.me  
3. **Track Status**: Webhooks automatically update status
4. **Manage Transactions**: Use wizard to capture, cancel or refund
