1. Instale o módulo através do Apps do Odoo
2. Configure suas credenciais do Pagar.me em Configurações > Provedores de Pagamento
3. Crie uma Configuração CNAB com processador "Pagar.me"
4. Configure os métodos de pagamento desejados (cartão, boleto, PIX)
5. Configure webhooks para atualizações automáticas
6. Teste a conexão com a API do Pagar.me
7. Configure modos de pagamento usando a configuração CNAB criada
